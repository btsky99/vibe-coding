# 📝 구현 계획 작성

**원칙: 계획 승인 전 코드 작성 금지.**

## 절차

1. **코드베이스 스캔**: 수정할 파일들의 현재 상태 파악
2. **태스크 분해**: 각 태스크는 30분 내 완료 가능한 크기로

각 태스크 형식:
```
[ ] Task N: {동사} + {대상} + {목적}
    파일: {경로}
    방법: {구체적인 구현 방법}
    검증: {완료 확인 방법}
```

3. **의존성 명시**: "Task 3은 Task 1 완료 후 시작" 형식
4. **계획 저장**: `ai_monitor_plan.md`에 저장
5. **승인 요청**: 사용자 확인 후 `/vibe:execute-plan`으로 이동
