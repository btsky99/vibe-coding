# 🧬 알려진 반복 패턴 DB

이 파일은 vibe-heal이 감지하고 치유한 반복 패턴을 누적 기록합니다.
새로운 패턴 치유 시 여기에 추가하세요.

---

## 패턴 목록

### P001 — rgb 관련 요청 반복 (6회)
- **증상**: 색상 관련 코드에서 반복 수정 요청
- **원인 유형**: 코드 결함
- **감지일**: 2026-03-06
- **치유 방법**: 색상 변수 중앙화 (CSS 변수 또는 상수 파일)

### P002 — 포트 번호 혼동 (8000 vs 9000 vs 19097)
- **증상**: 서버 포트를 잘못 참조하여 연결 실패
- **원인 유형**: 지식 부재
- **감지일**: 2026-03-13
- **치유 방법**: 개발 버전 기본 포트 = 9000, 서버는 경로 해시 기반 동적 포트(19xxx) 사용
- **재발 방지**: 이 패턴 파일에 기록 + vibe-orchestrate 스킬에 포트 지식 추가

### P003 — task_logs.jsonl 동시성 문제
- **증상**: 여러 에이전트 동시 쓰기 시 파일 손상
- **원인 유형**: 코드 결함
- **감지일**: 2026-03-01
- **치유 방법**: PostgreSQL pg_messages 테이블로 이전 (파일 기반 → DB)
- **상태**: 부분 해결 (messages.jsonl은 아직 파일 기반)
